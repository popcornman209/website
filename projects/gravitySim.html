Gravity Simulator

This is just a simple gravity simulator made in python from scratch, it just uses the turtle library for graphics. The main part is just a module that you can import, and really easily use. I did this as I was making a bunch of versions, some with collisions and others not, and I didnt want to have to code it every time.

Later I made a simple version of it in unity, which allowed a much better way to controll the camera (as the turtle library is... limited to say the least) and also a easier collision system.
